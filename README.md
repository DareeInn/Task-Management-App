# Task Management App

A Flutter task management app connected to Firebase Firestore with real-time synchronization.

## Features
- Add tasks
- View tasks in real time using StreamBuilder
- Toggle task completion
- Delete tasks
- Nested subtasks with add, toggle, and delete support

## Enhanced Features
1. Search / filter  
   Users can search tasks in real time by title, which makes it easier to find tasks quickly.

2. Delete confirmation dialog  
   Before deleting a task, the app asks for confirmation to prevent accidental removal.

## Firebase
This app uses Firebase Firestore for cloud storage and real-time task updates.

## Setup
1. Run `flutter pub get`
2. Configure Firebase with FlutterFire
3. Run the app with `flutter run`
